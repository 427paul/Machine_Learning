#1. 인공지능의 개념

지능 :
-오직 인간만 가지고 있는 고유의 성질
-무언가를 이해하고 배우는 능력
-본능적으로 행동하는 것이 아니라 생각하고 이해함으로써 행동으로 옮기는 능력

인공지능 :
-컴퓨터가 학습하고 생각하여 스스로 판단할 수 있도록 만드는 기술
사전적 개념 :
-철학적 개념
-지성을 갖춘 존재 또는 시스템에 의해 만들어진 인공지능을 의미
전통적 개념 :
-컴퓨터가 인간의 행동을 모방할 수 있도록 하는 소프트웨어
-인간이 가진 지적 능력의 일부 또는 전체를 구현한 것
기술적 개념 :
-인간의 지능으로 할 수 있는 사고, 학습, 자기계발 등을 컴퓨터가 할 수 있도록 하는 방법을 연구하는 컴퓨터공학 및 정보기술의 분야

프로그래밍 :
-발생할 수 있는 모든 케이스에 대해 사람이 프로그램으로 구현하고 행동
인공지능 :
-컴퓨터가 스스로 데이터를 학습하여 판단하고 행동


#2. 인공지능의 목표

다트머스 컨퍼런스 :
-1956년 미국 다트머스 대학교에서 10명의 학자들이 참여한 회의 -> 인공지능이라는 용어가 대중에게 처음 알려짐
전제 :
-학습의 모든 측면, 혹은 지능의 모든 특성이 원칙적으로 정확히 기술되어서 이를 모사하는 기계를 만들 수 있다는 가정에 토대를 두고 연구를 진행할 것
목표 :
-사람의 다양한 능력을 컴퓨터가 대신할 수 있도록 하는 것

인공지능 :
-미국의 전산학자이며, 인지학자인 존 맥카시가 만들었음.
-존 맥카시는 다트머스 컨퍼런스 개최를 준비하면서 경비 마련을 위해 록펠러 재단에 후원금 요청서를 보냈는데 이 요청서에 '인공지능'이라는 용어를 처음 사용한 것
-그는 인공지능에 대한 연구업적을 인정받아 1971년 튜링상을 수상함

인공지능 기술 : ['생활가전', '의료', '교육', '국방', '제조', '재무분석', '서비스'] 다양한 분야에서 활용되고 있음

인공지능의 발달 :
-로봇의 기능을 향상시키고 실생활에 로못이 도입될 수 있도록 하는 데 있음
-관련 예) 동영상 분석 기술 : 미래 인공지능 로봇의 눈이 제 역할을 할 수 있도록 발전시키는 과정에서 나타난 결과임


#3. 인공지능의 순기능

인공지능 기술 + 다양한 산업분야 -> 지금까지 상상할 수 없었던 새로운 결과물이 선보여지고 있음 : ['질병 진단 및 예측', '통/번역 서비스', '가상 개인 비서']

1. 노동대체 :
-인간의 정신적/육체적 노동을 최소화하거나 대체함으로써 업무 효율성을 크게 높일 수 있음
-관련 예) 아마존 '키바'로봇

-로못 바리스타가 등장하면서 향후 카페의 노동력을 대체할 것으로 예측됨

2. 자동화 :
-대표적인 RPA 서비스 : 보험금 청구 심사 시스템
-관련 예) 실손의료보험 청구 간소화 시스템
소비자(피보험자) ->진료비 납부 및 보험사에 서류 전송 요청 -> 병원 -> 서류 전송 위탁 -> 건강보험심사평가원 또는 전문 중계기관 -> 서류전송
->보험회사(인공지능으로 대체 -> 보험심사 업부 AI) -> 보험금 지급 -> 소비자(피보험자)
소비자(피보험자) ->보험금 청구서만 제출 -> 보험회사(인공지능으로 대체 -> 보험심사 업부 AI) -> 보험금 지급 -> 소비자(피보험자)
-보험심사 업무가 인공지능으로 자동화되면서 사람은 단순하고 반복적인 업무에서 해방되었고, 심사 업무는 빠르고 효율적으로 변화됨

RPA(Robotic Process Automation) : 사람이 수행하던 규칙적이고 반복적인 업무 프로세스에 소프트웨어 로봇을 적용하여 자동화하는 것
-저렴한 비용으로 빠르고 정확하게 업무를 수행하는 '디지털 노동'을 의미함
-사람 70명을 투입하여 15,628시간 동안 진행하던 업무를 인공지능이 접목된 RPA 1대로 대체 가능함

-협동로봇을 이용하면 제조 공정의 대부분을 자동화할 수 있음
-사람 : 위험한 작업환경에서 해방
-기업 : 높은 생산성 기대

3. 개인 능력 격차 완화 :
-인공지능은 다년간의 경험과 깊이 있는 지식을 보유한 전문가만이 할 수 있었던 업무를 비전문가도 쉽게 수행할 수 있도록 도와줌
-인공지능, 빅데이터 기술이 접목된 '로보어드바이저(Robo-advisor)'를 이용하면 투자 전문가가 아니더라도 고객에게 최적화된 투자자문 서비스를 제공할 수 있음


#4. 인공지능의 역기능

1. 멍청한 AI(Dumb AI) :
-인공지능에 악의적인 정보를 제공하면 잘못된 판단을 하거나 관점이 편향될 수 있음
-관련 예) MS의 인공지능 챗못 테이(Tay) : 일부 이용자들이 악의적인 정보를 제공하자, 이를 학습한 테이가 인종차별적 발언과 욕설을 쏟아내 차례

2. 법적/윤리적 책임 소재 문제 :
-인공지능은 현행법상 법적 권리나 의무의 주체인 사람(법인)이 아니므로 '법적 책임을 물을 수 없음'
-관련 예1) 자율주행차가 사고를 낸 경우, 배상책임과 처벌 문제
-관련 예2) 인공지능에 의한 질병검진과 로봇수술 허용 여부 등

3. 빈부격차 심화 :
-인공지능이 인간의 일자리를 대체함으로써 '인간의 생계 위협 가능성'
-인공지능 활성화로 새로운 작업이 창출되기도 하지만, 새로운 기술을 습득하고 활용하는 데에는 많은 시간 필요
-시간적, 기술 습득상의 제약으로 인해 배움에서 소외된 사람들은 경제적 어려움을 겪을 수 있음

4. 예측 불가능한 AI :
-학습되지 않았거나 추론을 할 수 없는 상황이 발생한 경우, 인공지능은 평소와 달리 엉뚱한 결과를 내놓을 수 있음
-관련 예) 바둑기사 이세돌 vs 알파고와의 대국 : 상대(이세돌)가 유리한 곳에 수를 두자 알파고가 실수를 연발하기 시작한 것
-> 알파고는 새로운 변수에 대응하지 못하는 기술적인 한계를 드러냄


# 인공지능의 중요성 

인공지능 : 중요환 화두
빅데이터 : '데이터양의 폭발적인 증가'가 한 몫함

기존(보유하고 있는 데이터) + 사물인터넷(수집된 데이터) -> 비즈니스에 활용하고자 하는 요구가 많아짐

최근 인공지능 기술 적용(['의료', 'IT산업', '제조', '금융']) -> '전산업'에 적용되고 이시는 추세

IT : SW 분석 솔류션

부동산 : 부동산 마케팅 솔루션

교육 : 온라인 교육의 혁명
